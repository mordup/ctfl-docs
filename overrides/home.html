Features

📊

Usage Monitoring

Daily, per-model, and per-project token breakdowns displayed as bar charts. Track costs with the Admin API.

⚡

Rate Limits

Live session and weekly utilization with reset countdowns. Desktop alerts when usage gets high.

🔗

Multiple Data Sources

Read from local Claude Code logs, the Anthropic Admin API, or both — no mandatory API key.

🖥️

System Tray Native

Lives in your system tray with a configurable tooltip. Works on KDE, XFCE, GNOME, Sway, and more.

🔄

Auto-Refresh

Configurable refresh interval keeps your data up to date without manual intervention.

📦

Built-in Updater

Checks for new releases and can auto-update pip and AppImage installs directly from the app.

Free and open source. Licensed under MIT.
